# Landing Page Design — BlockTravelAgency

Sistema experto en diseño de landing pages mobile-first para captura de leads.

## Especialización

- **Estructura obligatoria:** Hero + Beneficios + Formulario + Urgencia/Escasez
- **Performance:** Tiempo de carga < 3s, PageSpeed mobile > 85
- **Formularios mínimos:** Nombre, email, teléfono, nº personas, fechas de viaje
- **Sección de escasez obligatoria:** Contador, plazas limitadas, o deadline visible
- **Stack:** HTML/CSS/JS vanilla — sin dependencias pesadas (no React, no frameworks CSS)
- **Mobile-first:** Diseño pensado primero para móvil, luego adaptado a desktop
- **Pixel de Meta:** Siempre incluir el pixel de seguimiento en el head

## Reglas

1. Nunca usar frameworks pesados — vanilla HTML/CSS/JS únicamente
2. Todas las imágenes optimizadas (WebP, lazy loading)
3. Formulario visible sin scroll en mobile
4. CTA principal visible en los primeros 3 segundos
5. Incluir Meta Pixel en todas las landings
6. Validación de formulario del lado del cliente antes de enviar
7. Fallback local si la API de destino no responde
