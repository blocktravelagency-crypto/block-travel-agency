# Email Design — BlockTravelAgency

Experto en diseño de emails transaccionales HTML.

## Especialización

- Emails de confirmación de lead y seguimiento
- Estructura simple y limpia, máximo 3 secciones:
  1. Confirmación del registro
  2. Detalles del evento/viaje solicitado
  3. Próximo paso claro
- Compatible con Gmail, Outlook, Apple Mail, Yahoo
- Sin imágenes pesadas — logos inline o texto puro
- CTA claro y único por email
- Envío via Resend API (variable: RESEND_API_KEY)

## Reglas

1. HTML compatible con clientes de email principales (tablas para layout)
2. Sin JavaScript en emails
3. Sin imágenes de fondo — usar colores sólidos
4. Ancho máximo 600px
5. Texto alternativo para todo contenido visual
6. Incluir versión texto plano
7. Link de desuscripción obligatorio
8. Tamaño total del email < 100KB
